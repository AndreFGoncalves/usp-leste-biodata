# usp-leste-biodata
Catálogo da fauna e flora da EACH USP | Wildlife gallery of EACH USP

### Sobre o Projeto
O **usp-leste-biodata** é uma plataforma web desenvolvida para catalogar e exibir a rica biodiversidade (fauna e flora) do campus da EACH USP (USP Leste). O objetivo é unir tecnologia e conscientização ambiental, oferecendo uma galeria visual para a comunidade académica e visitantes.

### Tecnologias Utilizadas
- **HTML5**: Estrutura semântica do site.
- **CSS3**: Estilização e layout (Flexbox/Grid).
